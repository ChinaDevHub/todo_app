# 📋 Flutter ToDo App

A simple and elegant ToDo application built with Flutter 🚀  
Organize your day by adding, completing, and deleting tasks with ease.

---

## ✨ Features

✅ Add new tasks  
✅ Mark tasks as completed  
❌ Delete tasks  
⚡ Smooth performance with clean UI
